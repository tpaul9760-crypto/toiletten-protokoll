# Toiletten-Protokoll Normal-Version v1.4.1

Abgespeckte Version für normalen Gebrauch.

## Enthalten
- Kein Login
- Keine Firebase
- lokale Speicherung auf dem Gerät
- Mobile Kartenansicht
- Fotoaufnahme / Galerie / Dateien / Drag & Drop
- Export/Import inkl. Fotos
- Update-Prüfung
- Changelog als Dropdown
- PWA-fähig für Android/iPad

## Hinweis
Die Daten bleiben auf dem jeweiligen Gerät. Austausch erfolgt per Sicherung exportieren/importieren.
