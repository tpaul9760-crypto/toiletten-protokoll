SCHULTOILETTEN PROTOKOLL - HANDY APP / PWA

Schultoiletten-Protokoll

Eine einfache und kostenlose Web-App zur Dokumentation von Vandalismus, Schäden, Verstopfungen und Überschwemmungen in Schultoiletten.

Funktionen
Erfassung von Vorfällen mit Datum und Uhrzeit
Dokumentation von Schäden und Maßnahmen
Foto-Upload mit Mehrfachauswahl
Drag & Drop für Bilder
Großansicht der Fotos
Nachträgliches Bearbeiten von Einträgen
Suchfunktion für vorhandene Einträge
Druckfunktion für Protokolle
Export und Sicherung der Daten
Optimiert für PC, Tablet und Smartphone
Einsatzbereich

Die Anwendung wurde speziell für Hausmeister, Haustechnik, Reinigungskräfte und Schulleitungen entwickelt, um Schäden und Vandalismus auf Schultoiletten einfach und nachvollziehbar zu dokumentieren.

Datenschutz

Die Daten werden lokal im Browser des jeweiligen Geräts gespeichert. Es werden keine Daten an externe Server übertragen.

Hinweis

Fotos sollten ausschließlich zur Dokumentation von Schäden und nicht zur Aufnahme von Personen verwendet werden.

Lizenz

Kostenlose Nutzung auf eigene Verantwortung.

So nutzt du es:

ANDROID / CHROME:
1. Die Seite im Chrome-Browser öffnen.
2. Menü mit den drei Punkten öffnen.
3. „Zum Startbildschirm hinzufügen“ oder „App installieren“ auswählen.

IPHONE / SAFARI:
1. Die Seite in Safari öffnen.
2. Teilen-Symbol öffnen.
3. „Zum Home-Bildschirm“ auswählen.

WICHTIG:
- Damit die App-Installation richtig funktioniert, sollte die index.html über eine Webseite oder einen kleinen lokalen Server geöffnet werden.
- Direkt aus dem Dateiordner funktioniert die Startbildschirm-App je nach Handy/Browser nicht immer.
- Die Daten werden lokal auf dem jeweiligen Gerät gespeichert.
- PC, Handy und Kollege haben getrennte Einträge, solange kein gemeinsamer Server/Cloudspeicher genutzt wird.
- Fotos bitte nur von Schäden machen, nicht von Schülern.
